## 🤔 Bài toán đường đi ngắn nhất

Hãy tưởng tượng bạn dùng **Google Maps** tìm đường từ nhà đến công ty. Có nhiều tuyến đường, mỗi tuyến có **khoảng cách** (hoặc thời gian) khác nhau. Bạn muốn tìm **đường ngắn nhất**!

Đây chính là bài toán **Shortest Path** — và Dijkstra & Bellman-Ford là 2 giải thuật giải nó.

```
        A ──5──→ B
        │       ↗ │
       2│    3    │4
        │  ↗     ↓
        C ──1──→ D

Tìm đường ngắn nhất từ A đến D:
- A→B→D = 5 + 4 = 9
- A→C→D = 2 + 1 = 3 ✅ (ngắn nhất!)
- A→C→B→D = 2 + 3 + 4 = 9
```

---

## 🔷 Dijkstra's Algorithm

### Nguyên lý: Relaxation (Nới lỏng)

Gán khoảng cách ban đầu = ∞ cho tất cả, trừ nguồn = 0. Mỗi bước cập nhật khoảng cách **ngắn hơn** nếu tìm được.

### Cách hoạt động

1. Bắt đầu từ nguồn (source), khoảng cách = 0
2. Chọn node **chưa thăm** có khoảng cách **nhỏ nhất**
3. Cập nhật khoảng cách đến các hàng xóm (relaxation)
4. Đánh dấu đã thăm, lặp lại cho đến khi duyệt hết

```
Đỉnh:    A   B   C   D
Ban đầu:  0   ∞   ∞   ∞

Thăm A: B=5, C=2        → [0, 5, 2, ∞]
Thăm C: B=min(5,5)=5, D=3 → [0, 5, 2, 3]
Thăm D: (không cập nhật)  → [0, 5, 2, 3]
Thăm B: (không cập nhật)  → [0, 5, 2, 3]

Đường ngắn nhất từ A: B=5, C=2, D=3
```

### ⏱️ Time Complexity

| Implementation | Big O |
|----------------|-------|
| Array đơn giản | O(V²) |
| Priority Queue (Min Heap) | O((V + E) log V) ✨ |

> ⚠️ **Hạn chế**: Dijkstra **không hoạt động** với cạnh có **trọng số âm**!

## 🔶 Bellman-Ford Algorithm

### Khác gì Dijkstra?

- ✅ **Hoạt động với trọng số âm** (negative weights)
- ✅ Phát hiện **chu trình âm** (negative cycle)
- ❌ Chậm hơn Dijkstra: O(V × E)

### Cách hoạt động

1. Gán khoảng cách ban đầu = ∞, nguồn = 0
2. Lặp lại **V-1 lần**: duyệt qua **tất cả cạnh**, relax nếu tìm đường ngắn hơn
3. Lặp thêm lần thứ V: nếu vẫn relax được → có **chu trình âm**!

## 💻 Code mẫu: Dijkstra

### JavaScript

```javascript
// ===== DIJKSTRA - JAVASCRIPT =====

class PriorityQueue {
    constructor() { this.values = []; }
    enqueue(val, priority) {
        this.values.push({ val, priority });
        this.sort();
    }
    dequeue() { return this.values.shift(); }
    sort() { this.values.sort((a, b) => a.priority - b.priority); }
    isEmpty() { return this.values.length === 0; }
}

function dijkstra(graph, start) {
    const distances = {};
    const previous = {};
    const pq = new PriorityQueue();

    // Khởi tạo: tất cả = ∞, nguồn = 0
    for (const vertex in graph) {
        distances[vertex] = vertex === start ? 0 : Infinity;
        previous[vertex] = null;
        pq.enqueue(vertex, distances[vertex]);
    }

    while (!pq.isEmpty()) {
        const { val: current } = pq.dequeue();

        for (const { node: neighbor, weight } of graph[current]) {
            const newDist = distances[current] + weight;

            // Relaxation: tìm đường ngắn hơn?
            if (newDist < distances[neighbor]) {
                distances[neighbor] = newDist;
                previous[neighbor] = current;
                pq.enqueue(neighbor, newDist);
            }
        }
    }

    return { distances, previous };
}

// Truy vết đường đi
function getPath(previous, target) {
    const path = [];
    let current = target;
    while (current !== null) {
        path.unshift(current);
        current = previous[current];
    }
    return path;
}

// --- Test ---
const graph = {
    A: [{ node: "B", weight: 5 }, { node: "C", weight: 2 }],
    B: [{ node: "D", weight: 4 }],
    C: [{ node: "B", weight: 3 }, { node: "D", weight: 1 }],
    D: [],
};

const { distances, previous } = dijkstra(graph, "A");
console.log("Khoảng cách:", distances);
// { A: 0, B: 5, C: 2, D: 3 }

console.log("Đường A→D:", getPath(previous, "D"));
// ["A", "C", "D"]
```

### Python

```python
# ===== DIJKSTRA - PYTHON =====
import heapq

def dijkstra(graph, start):
    distances = {v: float('inf') for v in graph}
    distances[start] = 0
    previous = {v: None for v in graph}
    pq = [(0, start)]  # (distance, vertex)

    while pq:
        curr_dist, current = heapq.heappop(pq)

        if curr_dist > distances[current]:
            continue  # Bỏ qua nếu đã tìm đường ngắn hơn

        for neighbor, weight in graph[current]:
            new_dist = curr_dist + weight

            if new_dist < distances[neighbor]:
                distances[neighbor] = new_dist
                previous[neighbor] = current
                heapq.heappush(pq, (new_dist, neighbor))

    return distances, previous

def get_path(previous, target):
    path = []
    current = target
    while current is not None:
        path.append(current)
        current = previous[current]
    return path[::-1]

# --- Test ---
graph = {
    "A": [("B", 5), ("C", 2)],
    "B": [("D", 4)],
    "C": [("B", 3), ("D", 1)],
    "D": [],
}

distances, previous = dijkstra(graph, "A")
print("Khoảng cách:", distances)
# {'A': 0, 'B': 5, 'C': 2, 'D': 3}

print("Đường A→D:", get_path(previous, "D"))
# ['A', 'C', 'D']
```

### Rust

```rust
// ===== DIJKSTRA - RUST =====
use std::collections::{BinaryHeap, HashMap};
use std::cmp::Reverse;

fn dijkstra(graph: &HashMap<&str, Vec<(&str, i32)>>, start: &str) -> HashMap<String, i32> {
    let mut distances: HashMap<String, i32> = HashMap::new();
    let mut heap = BinaryHeap::new();

    for &vertex in graph.keys() {
        distances.insert(vertex.to_string(), i32::MAX);
    }
    distances.insert(start.to_string(), 0);
    heap.push(Reverse((0, start.to_string())));

    while let Some(Reverse((dist, current))) = heap.pop() {
        if dist > *distances.get(&current).unwrap_or(&i32::MAX) {
            continue;
        }

        if let Some(neighbors) = graph.get(current.as_str()) {
            for &(neighbor, weight) in neighbors {
                let new_dist = dist + weight;
                if new_dist < *distances.get(neighbor).unwrap_or(&i32::MAX) {
                    distances.insert(neighbor.to_string(), new_dist);
                    heap.push(Reverse((new_dist, neighbor.to_string())));
                }
            }
        }
    }

    distances
}

fn main() {
    let mut graph: HashMap<&str, Vec<(&str, i32)>> = HashMap::new();
    graph.insert("A", vec![("B", 5), ("C", 2)]);
    graph.insert("B", vec![("D", 4)]);
    graph.insert("C", vec![("B", 3), ("D", 1)]);
    graph.insert("D", vec![]);

    let distances = dijkstra(&graph, "A");
    println!("Khoảng cách: {:?}", distances);
    // {"A": 0, "B": 5, "C": 2, "D": 3}
}
```

## 🔄 Dijkstra vs Bellman-Ford

| Tiêu chí | Dijkstra | Bellman-Ford |
|----------|---------|-------------|
| Time | O((V+E) log V) ✨ | O(V × E) |
| Cạnh âm | ❌ Không hỗ trợ | ✅ Hỗ trợ |
| Phát hiện chu trình âm | ❌ | ✅ |
| Greedy | ✅ Có | ❌ Không |
| Dùng khi | Trọng số dương | Có trọng số âm |

---

## 🌍 Ứng dụng thực tế

| Tình huống | Giải thuật |
|------------|-----------|
| **Google Maps** | Dijkstra (trọng số = khoảng cách/thời gian) |
| **Network routing** | Bellman-Ford (OSPF, RIP protocol) |
| **Game pathfinding** | Dijkstra + A* (phiên bản nâng cao) |
| **Tài chính** (phát hiện arbitrage) | Bellman-Ford (chu trình âm = cơ hội lợi nhuận) |
| **GPS navigation** | Dijkstra |

---

## 🎯 Chọn giải thuật nào?

- **Trọng số dương only** → **Dijkstra** (nhanh hơn)
- **Có trọng số âm** → **Bellman-Ford**
- **Cần phát hiện chu trình âm** → **Bellman-Ford**
- **Graph thưa (sparse)** → Dijkstra + Min Heap
- **Graph dày (dense)** → Dijkstra + Array
