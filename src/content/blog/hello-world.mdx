Chào mừng bạn đến với Blog! Đây là bài viết mẫu được tạo bằng Markdown.
Bạn có thể dễ dàng sửa nội dung này thông qua giao diện `/admin` (TinaCMS).

<Callout title="Tin vui" type="success">
  Trang web của bạn đã tích hợp thành công React Components vào bài viết Markdown (MDX)!
</Callout>

Tính năng nổi bật:
- Hiệu năng tối đa (100% Static HTML).
- Visual Editing (Sửa trực tiếp trên trang).
- Tailwind CSS styling.
