## 🏗️ 1. Mẫu Khởi Tạo (Creational Patterns)

### 👑 Singleton (Duy nhât 1 Instance)
**Khi nào dùng?** App chỉ cần 1 Logger, 1 Cấu hình Database. Dùng đi dùng lại khắp mọi file.
```typescript
class Database {
  private static instance: Database;
  
  // Khóa constructor lại không cho thằng khác xài từ khóa `new`
  private constructor() { console.log("Connect to DB..."); }
  
  // Cửa duy nhất lấy ra xài cục CSDL này
  static getInstance(): Database {
    if (!Database.instance) {
      Database.instance = new Database(); // Chỉ new đúng 1 lần
    }
    return Database.instance;
  }
  
  query(sql: string) { console.log(`Executing: ${sql}`); }
}

const db1 = Database.getInstance(); // Hiện "Connect to DB..."
const db2 = Database.getInstance(); // Im ru (Đã lấy sẵn instance cũ)
```

### 🏭 Factory Method (Xưởng Sản Xuất)
**Khi nào dùng?** Khi bạn muốn khởi tạo object phức tạp bằng 1 dòng lệnh siêu gọn.
```typescript
interface Button { render(): void; }
class WindowsButton implements Button { render() { console.log("Hộp thoại Windows"); } }
class WebButton implements Button { render() { console.log("Nút HTML Canvas"); } }

// Abstract Factory: Thằng sếp, đưa ra chỉ thị "hãy làm nút"
abstract class Dialog {
  abstract createButton(): Button; // Chi tiết tạo nút để cấp dưới lo
  
  renderBtn() {
    const btn = this.createButton();
    btn.render();
  }
}

class WindowsDialog extends Dialog {
  createButton(): Button { return new WindowsButton(); }
}
class WebDialog extends Dialog {
  createButton(): Button { return new WebButton(); }
}

// Chạy:
new WebDialog().renderBtn(); // Nút HTML Canvas
```

### 🏢 Abstract Factory (Xưởng Đúc Đồng Bộ)
**Khi nào dùng?** Khách hàng mua bàn phải đi kèm ghế, mua Tủ phong cách Zen phải đi kèm Giường Zen. Tránh tình trạng nhét ghế Phục Hưng vào Bàn phong cách Alien. Mẫu này gom Đám Factory Method lại đẻ ra 1 bộ sậu chuẩn chỉ!
```typescript
// Các món đồ (Ghế & Bàn tròn dài tùy ý)
interface Chair { hasLegs(): boolean; }
interface Sofa { sitOnIt(): void; }

// Các Dòng Sản Phẩm (Gỗ xịn ngọc am vs Nhựa chợ lớn)
class VictorianChair implements Chair { hasLegs() { return true; } }
class VictorianSofa implements Sofa { sitOnIt() { console.log('Ngồi Êm Ru Kiểu Quý Tộc'); } }

class PlasticChair implements Chair { hasLegs() { return false; } }
class PlasticSofa implements Sofa { sitOnIt() { console.log('Ngồi gãy xương chậu'); } }

// ÔNG VUA XƯỞNG (Abstract Factory): Bắt buộc đẻ ra 1 cặp!
interface FurnitureFactory {
  createChair(): Chair;
  createSofa(): Sofa;
}

// Xưởng chuyên mài Đồ Cổ
class VictorianFactory implements FurnitureFactory {
  createChair() { return new VictorianChair(); }
  createSofa() { return new VictorianSofa(); }
}

// Dùng thật: Chốt đơn 1 Cặp Quý Tộc
const xuyenViet = new VictorianFactory();
const ghe = xuyenViet.createChair();
const sofa = xuyenViet.createSofa();
sofa.sitOnIt(); // Ngồi Êm Ru Kiểu Quý Tộc
```

### 🧱 Builder (Gã Thợ Lắp Ráp)
**Khi nào dùng?** Construct 1 object mà nó có tới 10 tham số. Nhìn dòng `new House(true, false, true, 4, 2, "gỗ")` là rùng mình. Trực giác méo biết tham số thứ 3 `true` là bể bơi hay là chó giữ nhà?? Có Thợ Lắp Ráp thì ta xây từng khối theo tên đàng hoàng.
```typescript
class House {
  public walls = 0;
  public doors = 0;
  public hasPool = false;
  public hasGarage = false;
}

// Thằng Thợ Hồ (Chỉ tuân lệnh, làm xong nhả Nhà ra)
class HouseBuilder {
  private house = new House();
  
  buildWalls(n: number) { this.house.walls = n; return this; }
  buildDoors(n: number) { this.house.doors = n; return this; }
  buildPool() { this.house.hasPool = true; return this; }
  buildGarage() { this.house.hasGarage = true; return this; }
  
  getResult() { return this.house; }
}

// Triệu hồi Thợ:
const myLuxuryVilla = new HouseBuilder()
  .buildDoors(5)
  .buildWalls(20)
  .buildPool()
  // Không thích Garage thì méo gọi buildGarage, chả chết ai
  .getResult();
  
console.log(myLuxuryVilla); // { walls: 20, doors: 5, hasPool: true, hasGarage: false }
```

### 🐑 Prototype (Nhân Bản Cừu Dolly)
**Khi nào dùng?** Lấy Data từ Database tạo ra con Quái Vật Level 99 siêu to khổng lồ mất 5s Load RAM. Bạn muốn đẻ ra 1 vạn con quái giống hệt như vậy mà méo màng vào Database lấy nữa. Gọi thuật phân thân Chi Bóng Cừu Dolly!
```typescript
class Monster {
  constructor(public name: string, public hp: number, public secretWeapon: string) {
    // Siêu nặng, load ngầm API database DB mất 5s...
  }
  
  // Hàm nhân bản Vĩ Đại
  clone(): Monster {
    // Copy luôn bộ nhớ RAM không tốn công chọc lại Database
    return new Monster(this.name + " (Clone Thừa Kế)", this.hp, this.secretWeapon);
  }
}

const bossGiua = new Monster("Diablo", 9999, "Lửa Địa Ngục");
const deCuaBoss = bossGiua.clone(); // Load 0 giây
console.log(deCuaBoss.name); // Diablo (Clone Thừa Kế)
```

## 2. Mẫu Cấu Trúc (Structural Patterns)

### 🔌 Adapter (Kẻ chuyển đổi)
**Khi nào dùng?** API cũ trả về chuỗi XML, nhưng Chart Library UI mới lại chỉ xài mảng JSON. Ta chế một cái Adapter cắm vào giữa.

```typescript
// Món rởm
class XmlApi { getXml() { return "<user>Hoa</user>"; } }
// Đồ xịn cần dùng JSON
class JsonChartUI { render(jsonObj: any) { console.log("Vẽ mượt", jsonObj.name); } }

// 📌 Adapter cứu rỗi đời lập trình viên
class XmlToJsonAdapter {
  constructor(private xmlApi: XmlApi) {}
  
  getJson() {
    const rawXml = this.xmlApi.getXml();
    // (Logic Fake Regex) Cắt chuỗi để giả vờ chuyển XML -> JSON
    return { name: "Hoa" };
  }
}

// Xài thôi:
const oldSystem = new XmlApi();
const adapter = new XmlToJsonAdapter(oldSystem);

const ui = new JsonChartUI();
ui.render(adapter.getJson()); // Vẽ mượt Hoa
```

### 👘 Decorator (Bọc Thêm Năng Lực)
**Khi nào dùng?** Bạn có class `Cafe`, người ta muốn mua `Cafe + Sữa`, người khác mua `Cafe + Bọt`, người dị dợm mua `Cafe + Sữa + Bọt x2`. Mẫu này giúp tính giá xịn xò.

```typescript
interface Coffee { cost(): number; }
class BlackCoffee implements Coffee { cost() { return 20; } }

// Bộ Bọc Trang Trí
class CoffeeDecorator implements Coffee {
  constructor(protected coffee: Coffee) {} // Chứa li cafe gốc bên trong
  cost() { return this.coffee.cost(); }
}

class MilkDecorator extends CoffeeDecorator {
  cost() { return super.cost() + 5; } // Thêm 5k tiền sữa
}

class WhipDecorator extends CoffeeDecorator {
  cost() { return super.cost() + 10; } // Thêm 10k tiền bọt
}

// Trải nghiệm khách hàng:
let myCup: Coffee = new BlackCoffee();
myCup = new MilkDecorator(myCup); // Chế sữa vào ly
myCup = new WhipDecorator(myCup); // Thêm bọt
console.log(myCup.cost()); // Ra lò: 20 + 5 + 10 = 35k
```

### 🎭 Facade (Mặt Nạ Đơn Giản Hóa)
**Khi nào dùng?** Bộ đồ lòng Máy Giặt bạn soi thấy là 1 đống Motor Xoay, Van Bơm Nước, Phễu xả... Khách lạ chỉ muốn "Bấm đúng 1 nút thì Áo Nhả Ra Giặt Ngon", méo màng quan tâm trong ruột cồng kềnh cỡ nào. Facade đi may cái Nút Giao Diện ảo ma che đi cái mớ bòng bong hắc ám ở sau đó.
```typescript
// Các thành phần ruột rà hỗn độn vứt tóe loe
class VanNuoc { moVan() { console.log('Nước chảy rào rào'); } }
class MotorTruc { quay(vang: number) { console.log(`Quay tốc độ ${vang} vòng`); } }
class CuaMayGiat { dongCua() { console.log('Chốt cửa cách ly cực ngặt!'); } }

// GÃ MẶT NẠ (Facade) Gánh Nợ Tội Lỗi
class MayGiatFacade {
  private van = new VanNuoc();
  private motor = new MotorTruc();
  private cua = new CuaMayGiat();

  giatSachQuanAo() {
    this.cua.dongCua();
    this.van.moVan();
    this.motor.quay(1000);
    console.log('Ting Ting! Đem phơi đi thằng lười!');
  }
}

// Bọn nghiệp dư móc ra xài húp trọn
const mayGiat = new MayGiatFacade();
mayGiat.giatSachQuanAo(); // 1 nút Bấm cân mọi thế giới
```

### 🚪 Proxy (Người Gác Cổng / Chặn Lỗ Hổng)
**Khi nào dùng?** App bạn có Class `Youtube` tải Video tốn 1 GB mạng. User Click Spam 20 chục phát nút Tải, server bị thắt cổ hốc máu. Lập 1 thằng Người Giới Đứng Trước, gõ kẻng: "Mày tải rôì, t tống cái Cache ra nhét vô mặt mày chứ không cấp Tải thực tế nữa".
```typescript
interface Downloader { download(videoUrl: string): string; }

// Cục API Thật (Nặng Chảy Chết mẹ)
class RealYoutube implements Downloader {
  download(videoUrl: string) {
    console.log(`Đang cút tận xuống Server Đáy Biển tải cục: ${videoUrl}`);
    return "Phim_Hanh_Dong.mp4";
  }
}

// Người Gác Cổng Kế Toán Xanh Chín
class ProxyYoutube implements Downloader {
  private realVideo = new RealYoutube();
  private cachedVideo: Record<string, string> = {}; // Tủ lạnh chứa đồ ăn thừa Cắm 1 Gb

  download(videoUrl: string) {
    if (!this.cachedVideo[videoUrl]) {
      // Non trẻ gọi lần đầu, Cho Đi Xúc đất làm đồ thật
      this.cachedVideo[videoUrl] = this.realVideo.download(videoUrl);
    } else {
      console.log(`[Ủa mải tải rồi mà?] - Quạt cái rầm Cache vào mặt: ${videoUrl}`);
    }
    return this.cachedVideo[videoUrl];
  }
}

// Demo Spam Kinh Hoàng Lũ Trẻ:
const bouncer = new ProxyYoutube();
bouncer.download("link_phim"); // Đi tải thật 1 phút
bouncer.download("link_phim"); // Rét! Nhè Cache 0ms 
bouncer.download("link_phim"); // Rét! Nhè Cache 0ms
```

### 🌉 Bridge (Chiếc Cầu Bẻ Vuông Tròn Tách Việc)
**Khi nào dùng?** Khi Code Cứng 2 nhánh Răng Cưa: Cấp Lớp "Cái Nút Đỏ", "Cái Nút Xanh" xong rồi khách bảo cho thêm Nút Cam t thành "Nút Cam", "Nút Lõm Xanh"?? Cấp độ Lai Nhân Bản bành trướng sập RAM (Combinatorial Explosion). Bridge đập cái búa Chia 2 vế rõ Ràng: Hình Đi Đằng Hình; Chó đi đằng Màu Mẻ.
```typescript
interface Color { fill(): string; }
class Red implements Color { fill() { return "Đỏ khè máu"; } }
class Blue implements Color { fill() { return "Xanh tái mét"; } }

// Cầu nối (Bridge) Cắm màu Rời Nuôi Băng Hình Tổ Tông
abstract class Shape {
  constructor(protected color: Color) {} // Cái Cán lỗ cắm cọ vẩy Tông Màu tùy chỉnh
  abstract draw(): void;
}

class Circle extends Shape {
  draw() { console.log(`Trổ hình Vòng Tròn màu: ${this.color.fill()}`); }
}
class Square extends Shape {
  draw() { console.log(`Đục Lỗ Vuông màu: ${this.color.fill()}`); }
}

// Nêm Nếm Quẩy:
const blueCircle = new Circle(new Blue());
blueCircle.draw(); // Vòng Hình Tròn Màu Xanh tái mét (Chia Rạch Lược Hoàn Mỹ)
```

### 🌳 Composite (Dây Thìa Canh Rễ Trùm / Búp Bê Nga)
**Khi nào dùng?** Lái Đóng Gói Một Thùng Hàng trong đó có 2 Gói Bưu Phẩm Bọc Trong. Trong Góc 1 Gói lại lấp Liếm 3 Hộp Gói Nhỏ lồng chéo. Tính Tổng Bill Lô Lô Hàng ra Bao Nhiêu Tiền Kg Vận Đơn? Buộc Đối Tượng Đơn Lẻ Mảnh (Khăn Tắm Đồng Giá) và Nhóm Ngấu Nghiến (Sọt Tôm Càng) vào Cùng Chung 1 Quy Tắc Get Tiền.
```typescript
interface Component { getPrice(): number; }

// Điểm Lót Đáy Cùng Hộp - Chấp Mọi Ngang Khía
class ProductItem implements Component {
  constructor(private name: string, private price: number) {}
  getPrice() { return this.price; }
}

// Túi Bọc Ngoại Diện (Chứa Con Nhỏ Trùm)
class BoxComposite implements Component {
  private boxChildren: Component[] = [];
  
  add(child: Component) { this.boxChildren.push(child); }
  
  // Ma thuật Đệ Quy (Cứ có Thằng Lỏi Con thì bọc cổ xâu lôi tiền lên Tổng)
  getPrice() {
    return this.boxChildren.reduce((sum, child) => sum + child.getPrice(), 0);
  }
}

// Cố Chốt Lô:
const hoTo = new BoxComposite(); // Rương Ngoài Cùng Bự Tổ Bố

const hopNho = new BoxComposite(); // Hộp nhét góc Thêm
hopNho.add(new ProductItem("Sạc Mac Trâu Bò", 1500));
hopNho.add(new ProductItem("Tai Nghe Rẻ Rách", 100));

hoTo.add(hopNho); // Hộp lại quăng Thủng Rương Hộp Bự
hoTo.add(new ProductItem("Sách Design Pattern Chân Kinh", 400)); // Lẻ tẻ Mắc cạn ngoài bìa rương

console.log(hoTo.getPrice()); // Tổng 2000 (Chạm tay là Tính hết Dòng Cọp Cuộn)
```

### 🪰 Flyweight (Hạt Cát Bay Tỏa Hồn RAM)
**Khi nào dùng?** Team Rừng Game đẻ ra 1 Triệu Cây Cổ Thụ. Nét Bể Texture Lá mỗi Cây Size tống 5MB. Tràn Đom Đóm Máy Người Chơi Bắt Lóe (5 Triệu MB - 5TB Rớt Nhịp). Cất tủ Mảng Xinh Đẻ ra Chỉ Đúng 1 Ảnh Lá Xanh Vẽ 5MB dùng Chăn Trâu Dùng Chung! Cây Chơi Toàn Map Chỉ Khác Nút Tọa Độ Cắm Đất (X,Y) Trắng Bóc Thôi.
```typescript
// Trạng Thái Nồi Sinh Nội Độc (Cục Kẽm Khổng Lồ Chia Chia Chung)
class TreeType {
  constructor(public modelName: string, public textureTonsOfMegabytes: string) {}
  render(x: number, y: number) { console.log(`Cắm Mọc Cây ${this.modelName} Tại Map Điểm (X: ${x}, Y: ${y})`); }
}

// Khố Chứa Chung Tối Giản Cache Factory (Đỉnh Cao Trừ Bì)
class TreeFactory {
  static treeTypes: Record<string, TreeType> = {};
  
  static getTreeType(name: string, tx: string) {
    if (!this.treeTypes[name]) {
      this.treeTypes[name] = new TreeType(name, tx); // Thiếu mới Build Hình Nấm
    }
    return this.treeTypes[name]; // Trả Cache Xanh Chín Trơn Láng
  }
}

// Cây Giấy Con Con Trên Đất Tốn Đúng 1KB Bộ Nhớ Trí Khôn
class Tree {
  constructor(private x: number, private y: number, private type: TreeType) {}
  draw() { this.type.render(this.x, this.y); }
}

// Rải 10 Vạn Triệu Hạt Cây Map Không Rớt 1 Lỗ Đen FPS Nào!
const loamSoi = TreeFactory.getTreeType("Cây Sồi Trăm Tuổi", "img_soi_nang_tram_mb.png");
const cay1 = new Tree(10, 20, loamSoi);
const cay2 = new Tree(99, 50, loamSoi); 
cay1.draw(); cay2.draw();
```

## 3. Mẫu Hành Vi (Behavioral Patterns)

### 📰 Observer (Chuông báo Event)
**Khi nào dùng?** Hệ thống Event lừng danh trong Javascript `(addEventListener)`.

```typescript
// Kênh Youtube
class Channel {
  private subs: ((video: string) => void)[] = [];
  
  subscribe(sub: (video: string) => void) { this.subs.push(sub); }
  
  uploadVideo(title: string) {
    console.log(`Đã ra mắt clip: ${title}`);
    this.subs.forEach(s => s(title)); // Bắn tin báo cho triệu con chim lợn
  }
}

// Người theo dõi (Observer)
const userLam = (vid: string) => console.log(`Lâm Đã vô xem ${vid} cmt đầu!`);
const userHoa = (vid: string) => console.log(`Hoa để mai coi ${vid}`);

const mixi = new Channel();
mixi.subscribe(userLam);
mixi.subscribe(userHoa);

mixi.uploadVideo("Review Bàn Phím");
// Kết quả (tự động 100%):
// Lâm Đã vô xem Review Bàn Phím cmt đầu!
// Hoa để mai coi Review Bàn Phím
```

### 🕹️ Strategy (Bỏ 5 món võ vào 1 cái túi)
**Khi nào dùng?** 1 game quái vật lúc bình thường đánh Melee, xuống 20% máu xài skill Fireball, sắp chết dùng skill Cắn lén. Thay vì check if-else máu, đổi luôn chiến thuật.

```typescript
// Các chiến thuật độc lập hoàn toàn với nhau
interface AttackStrategy { attack(): void; }
class MeleeAttack implements AttackStrategy { attack() { console.log('Vuốt móng vuốt!'); } }
class FireAttack implements AttackStrategy { attack() { console.log('Phun lửa!'); } }
class RunAway implements AttackStrategy { attack() { console.log('Chạy sút quần!'); } }

// Con quái chứa chiến thuật
class Boss {
  constructor(private strategy: AttackStrategy) {}
  
  // Đóng nóng (Hotswap) chiến thuật ngay giữa trận đấu!
  setStrategy(st: AttackStrategy) { this.strategy = st; }
  
  doAction() { this.strategy.attack(); }
}

const dragon = new Boss(new MeleeAttack());
dragon.doAction(); // Vuốt móng vuốt!

dragon.setStrategy(new FireAttack()); 
dragon.doAction(); // Sang Phase 2 đấm lửa rát mặt.

dragon.setStrategy(new RunAway());
dragon.doAction(); // Thấy nhân vật chính buff quá, lượn cho nước nó trong.
```

### 📜 Command (Lưu Lệnh Thành 1 Tờ Giấy Thỉnh Tội)
**Khi nào dùng?** App của sếp có nút Undo/Redo "Hủy thao tác" hoặc muốn Hẹn Giờ 1 tiếng sau mới chạy Lệnh Gửi Mail. Thay vì ấn phát chạy luôn, gói hành động đó lại thành 1 cái Thẻ Bài (Command) có Khấc Cài, ném cho thằng khác lúc nào thích thì xài.
```typescript
interface Command { execute(): void; undo(): void; }

// Thẻ Bài: Rút Gươm
class RutGuomCommand implements Command {
  execute() { console.log("Rút kiếm Nhật đánh Văng Đầu!"); }
  undo() { console.log("Cất kiếm vô vỏ cạch, Liếm Vết Máu!"); }
}

// Thẻ Bài: Bắn Cung
class BanCungCommand implements Command {
  execute() { console.log("Thả Cung Bay Chéo Trời Mây!"); }
  undo() { console.log("Chạy Ra Lụm Mũi Tên Về Quiver Cất Dấu"); } // Hoàn Tác Kinh điển
}

// Bảng Điều Khiển Tay Cầm Khống Chế
class GameController {
  private history: Command[] = []; // Nơi chứa Lịch sử Ấn Nút Mòn Dấu Tay
  
  bamNut(cmd: Command) {
    cmd.execute();
    this.history.push(cmd); // Lưu thẻ bài vào bộ Trí Nhớ Vừa Nhấp
  }
  
  bamNutZ() { // Ctrl+Z Undo Thần thánh Hoàn Tục Tội Lỗi Vụt Mất
    const lastCmd = this.history.pop();
    if (lastCmd) lastCmd.undo(); 
  }
}

const tayCam = new GameController();
```

```typescript
// Các Mảnh Ghép Xương Của Lệnh
interface PhienDich { dichRaNghia(): number; }

class ChuSo implements PhienDich {
  constructor(private so: number) {}
  dichRaNghia() { return this.so; }
}

// Thằng Cộng Dồn Răng Rụng
class PhepCong implements PhienDich {
  constructor(private trai: PhienDich, private phai: PhienDich) {}
  dichRaNghia() { return this.trai.dichRaNghia() + this.phai.dichRaNghia(); }
}

// Giả Phỏng 1 Máy Đọc Chuỗi Khách Vứt Vào ("1 + 5") Rồi Giải Quyết Nó Nhanh Nhọn:
const phepTinhToanThucThe = new PhepCong(new ChuSo(1), new ChuSo(5)); 
// Thằng Phân Tịch Ast Sinh Ra Cây Này Rồi Mới Lôi Đầu Ra Execute:
console.log(phepTinhToanThucThe.dichRaNghia()); // 6 Xa Nhấn Tựa Mũi Lao
```
